# Stock Finance project

A professional level financial analytics dashboard built with Python and Streamlit.

This project uses real time stock market data, technical indicators, sentiment analysis, currency conversion, and AI-powered financial Q&A into a single interactive platform inspired by tools like Google Finance and Bloomberg.


## Features

- Real-time stock price lookup
- Interactive candlestick charts
- Technical indicators (SMA, EMA, RSI, MACD, Bollinger Bands)
- Company financial overview
- News-based sentiment analysis
- AI-powered financial question answering
- Currency conversion tools
- Clean, professional Streamlit UI
- Mobile-friendly layout


## Technologies Used

- Python
- Streamlit
- Pandas / NumPy
- Plotly
- Financial APIs (real-time market data)
- Natural Language Processing (sentiment analysis)
- AI API for financial Q&A
